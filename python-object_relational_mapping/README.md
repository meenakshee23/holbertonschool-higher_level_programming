## Object-relational Mappers(ORMs)

- An Object-Relational Mapper (ORM) is a library that automatically converts data from database tables into objects used in application code.

ORMs are useful because they let developers interact with a database using Python code instead of writing SQL. This makes it easier to create, read, update, and delete data using a programming language they are already familiar with.
