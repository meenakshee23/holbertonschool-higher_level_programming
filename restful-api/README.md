### RESTful API
REST API Introduction


**difference between http and https in simple words**

- HTTP messages are plaintext, which means unauthorized parties can easily access and read them over the internet. In contrast, HTTPS transmits all data in encrypted form.

**Understand the basic working and structure of HTTP requests and responses.**

- An HTTP request and response together form the basic web communication process: a client sends a request (with a method, URL, headers, and sometimes a body), and the server replies with a response (containing a status code, headers, and the requested content).
